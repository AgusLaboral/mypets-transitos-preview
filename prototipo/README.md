# My Pets — Tránsitos y adopciones

Prototipo navegable de la extensión de My Pets para encontrar hogares de tránsito, revisar compatibilidad, acompañar adopciones y coordinar casos.

## Recorrido demostrable

1. Inicio con dos intenciones claras: buscar animales u ofrecer tránsito.
2. Búsqueda por necesidad y zona.
3. Perfil de Mora con urgencia, rutina y cuidados.
4. Compatibilidad en tres preguntas, una por pantalla.
5. Revisión, envío de disponibilidad y confirmación.
6. Mis casos y seguimiento post-adopción.
7. Vista operativa separada para el equipo My Pets.

El prototipo es estático: no persiste datos ni conecta servicios reales. Navegación, elecciones, estados de confirmación, descarga `.ics` y adaptación responsive sí funcionan para validar el producto antes del backend.
